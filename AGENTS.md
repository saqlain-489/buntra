# Buntra Web Studio

This project's full guidance for AI agents lives in **[CLAUDE.md](./CLAUDE.md)** — read it first.

It is the single source of truth. It covers:

- **Project goal:** a conversion landing page whose one job is turning visitors into free-mockup leads.
- **Stack & commands:** Next.js 15, React 19, TypeScript, Tailwind v4, GSAP, Node 22 via nvm.
- **Conventions:** copy in `lib/content.ts`, shared `primitives.tsx`, `Reveal` for motion, `@/*` alias, load-bearing anchor IDs.
- **The "Amber + Ink" design system** (tokens, fonts, radii, button/theme/color locks) and the **locked Hero**.
- **Design rules — human psychology & conversion** to apply to every section (risk reversal, loss aversion, trust-before-ask, honest proof, color psychology).
- **Design rules — craft / anti-slop** (no em-dashes, eyebrow restraint, no layout repetition, real images, WCAG AA).
- **Page structure** and **working rules** (analysis vs action, verify in a real browser).

Do not duplicate rules here; edit `CLAUDE.md` so the two never drift.
